# @equmanager/mobile

App nativa con Expo + React Native. Reutiliza `@equmanager/auth` y `@equmanager/domain` del monorepo.

**Estado actual:** esqueleto. La implementación real arranca en la **Fase 6** del plan (ver README raíz).

## Arrancar

```bash
pnpm --filter @equmanager/mobile dev
```

Para builds de producción se usa EAS (Expo Application Services). La configuración se añadirá cuando entremos en fase 6.
